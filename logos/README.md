# Company Logos

This folder contains logos for the companies featured in the portfolio.

## Required Logo Files

Place the following logo files in this directory:

1. `anbt-logo.png` - ANBT (National Agency of Dams and Transfers)
2. `engie-logo.png` - ENGIE
3. `cnr-logo.png` - Compagnie Nationale du Rhône
4. `reor20-logo.png` - REOR20
5. `cnrs-logo.png` - CNRS
6. `inrae-logo.png` - INRAE
7. `swa-logo.png` - Saudi Water Authority
8. `miyahthon-logo.png` - Miyahthon
9. `idws-logo.png` - IDWS

## Logo Specifications

- **Format**: PNG with transparent background
- **Size**: 150x80 pixels (width x height)
- **File size**: < 50KB per logo
- **Colors**: Original company colors

## How to Get the Logos

1. Visit each company's website
2. Look for their logo in the header/footer or press kit
3. Right-click and "Save Image As"
4. Use an online image optimizer like TinyPNG.com
5. Resize to 150x80px maintaining aspect ratio
6. Save as PNG with transparent background if possible
